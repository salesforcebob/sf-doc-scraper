# Agent Script Reference

This reference covers key concepts and terms associated with Agent Script.

-   **[Actions](/docs/einstein/genai/guide/ascript-ref-actions.html)** - Define executable tasks that an agent can perform, such as running a flow or transitioning to a new topic.
    
-   **[Before and After Reasoning](/docs/einstein/genai/guide/ascript-ref-before-after-reasoning.html)** - Optional blocks inside a topic that run either before or after the topic executes.
    
-   **[Blocks](/docs/einstein/genai/guide/ascript-blocks.html)** - The structural components of an Agent Script, where each block contains a set of properties that describe data or procedures.
    
-   **[Conditional Expressions](/docs/einstein/genai/guide/ascript-ref-expressions.html)** - Deterministically specify what actions to take or which prompts to include based on the current context.
    
-   **[Reasoning Instructions](/docs/einstein/genai/guide/ascript-ref-instructions.html)** - Instructions that Agentforce resolves into a prompt for the LLM.
    
-   **[Supported Operators](/docs/einstein/genai/guide/ascript-ref-operators.html)** - The comparison, logical, and arithmetic operators you can use in Agent Script.
    
-   **[Tools (Reasoning Actions)](/docs/einstein/genai/guide/ascript-ref-tools.html)** - Executable functions that the LLM can choose to call, based on the tool's description and current context.
    
-   **[Topics](/docs/einstein/genai/guide/ascript-blocks.html#topic-blocks)** - A set of instructions, actions, and reasoning that defines a job that an agent can do.
    
-   **[Utils](/docs/einstein/genai/guide/ascript-ref-utils.html)** - Utility functions used as tools, such as transitioning to topics or setting variable values.
    
-   **[Variables](/docs/einstein/genai/guide/ascript-ref-variables.html)** - Let agents track information across conversation turns.
