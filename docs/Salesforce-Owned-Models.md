# Salesforce-Owned Models

Salesforce AI Research creates, trains, and fine tunes models to address specific Salesforce use cases.

## CodeGen

CodeGen is a powerful language model capable of generating code.

[Agentforce Vibes Extension](https://developer.salesforce.com/docs/platform/einstein-for-devs/guide/einstein-overview.html) is an AI-powered developer tool that's available as an easy-to-install Visual Studio Code extension. It's built using CodeGen.
