# Android Configuration Overview

Configuration classes and patterns for the Agentforce Android SDK.

## Configuration Classes

-   **[AgentforceMode](/docs/einstein/genai/references/agentforce-mobile-sdk/agentforce-mode-android.html)** - Agent type configuration
-   **[AgentforceConfiguration](/docs/einstein/genai/references/agentforce-mobile-sdk/agentforce-configuration-android.html)** - Main configuration class
-   **[EmployeeAgentConfiguration](/docs/einstein/genai/references/agentforce-mobile-sdk/employee-agent-configuration-android.html)** - Employee agent settings
-   **[ServiceAgentConfiguration](/docs/einstein/genai/references/agentforce-mobile-sdk/service-agent-configuration-android.html)** - Service agent settings
-   **[AgentforceClient](/docs/einstein/genai/references/agentforce-mobile-sdk/agentforce-client-android.html)** - Client initialization
-   **[AgentforceConversation](/docs/einstein/genai/references/agentforce-mobile-sdk/agentforce-conversation-android.html)** - Conversation management
-   **[AgentforceContextManager](/docs/einstein/genai/references/agentforce-mobile-sdk/agentforce-context-manager-android.html)** - Context management

## Quick Start

1.  **Choose Agent Type**: Select between EmployeeAgent, ServiceAgent, or FullConfig
2.  **Configure Settings**: Set up authentication, UI preferences, and feature flags
3.  **Initialize Client**: Create and configure the AgentforceClient
4.  **Start Conversation**: Begin user interactions with the configured agent
