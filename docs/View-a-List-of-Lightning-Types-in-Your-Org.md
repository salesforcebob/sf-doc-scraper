# View a List of Lightning Types in Your Org

To view the custom and standard Lightning types in your org, from Setup, enter `Lightning Types` in the Quick Find box, and then select **Lightning Types**.

![The Lightning Types page shows a list of Lightning types in your org. The Source column indicates whether a Lightning type is standard or custom.](https://a.sfdcstatic.com/developer-website/sfdocs/genai/media/lightning-types-images/258-Updated-Setup-UI.png)

From this page, you can:

-   View all standard and custom Lightning types in your org.
-   View the details for a specific type by clicking its label.
-   Create a new custom Lightning type by clicking New Lightning type.
