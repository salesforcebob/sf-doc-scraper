# Core Classes

The Agentforce Mobile SDK for Android provides core classes that form the foundation of the SDK functionality.

## AgentforceClient

The main client class for initializing and managing the Agentforce SDK.

## AgentforceConversation

The conversation management class that handles chat interactions and conversation state.
