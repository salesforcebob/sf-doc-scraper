# Agentforce Mobile SDK Developer Guide

The Agentforce Mobile SDK empowers you to integrate Salesforce's feature-rich, trusted AI platform directly with your native iOS and Android mobile applications. By leveraging the Agentforce Mobile SDK, you can deliver cutting-edge, intelligent, and conversational AI experiences to your mobile users, enhancing engagement and providing seamless access to information and actions.

## Platform Support

The Agentforce Mobile SDK supports both iOS and Android platforms with feature parity.

-   **iOS**: Requires iOS 17+ with SwiftUI support
-   **Android**: Requires Android API level 29+ with Jetpack Compose support
