# Android UI Components

Native Jetpack Compose components for building conversational interfaces with the Agentforce Mobile SDK.

## Available Components

Build conversational interfaces with these native Compose components:

-   **[AgentforceComponent](/docs/einstein/genai/references/agentforce-mobile-sdk/agentforce-component-android.html)** - Base component system for UI rendering and interaction
-   **[ChatResponseButton](/docs/einstein/genai/references/agentforce-mobile-sdk/chat-response-button-android.html)** - Interactive response buttons and action handling
-   **[RecordSelector](/docs/einstein/genai/references/agentforce-mobile-sdk/record-selector-android.html)** - Salesforce record selection and display components

## Integration

All UI components follow Material Design principles and integrate with the Android theming system for consistent styling and customization.
