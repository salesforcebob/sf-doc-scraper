# Get Started With Enhanced Chat v2

Enhanced Chat v2 is a new customer interface for Enhanced Web Chat on Service Cloud. Both Enhanced Chat versions allow you to connect and deploy Service agents to external chat channels.

The UI of Enhanced Chat v2 can be customized with custom Lightning Types. See [Introduction to Lightning Types](/docs/einstein/genai/guide/lightning-types-get-started.html).

## Context Events

Enhanced Chat v2 offers a unique way to pass context events. `utilAPI` in Enhanced Web Chat now has a new `setSessionContext` method that can be used in Enhanced Chat v2 deployments. See [the enhanced chat developer guide](https://developer.salesforce.com/docs/service/messaging-web/guide/context-events.html) and [the enhanced chat reference](https://developer.salesforce.com/docs/service/messaging-web/references/m4w-reference/utilAPI.html#setsessioncontext) in Service Cloud.

## Inline Mode

You can embed the Enhanced Chat v2 client directly within a specific element on your web page. Learn how to enable and configure inline mode in [Inline Mode in Enhanced Chat v2](/docs/einstein/genai/guide/enhanced-chat-inline-mode.html).
