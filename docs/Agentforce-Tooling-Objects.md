# Agentforce Tooling Objects

Manage actions and customize the planner service using the [Tooling API](https://developer.salesforce.com/docs/atlas.en-us.api_tooling.meta/api_tooling/intro_api_tooling.htm).

-   [GenAiFunctionDefinition](https://developer.salesforce.com/docs/atlas.en-us.api_tooling.meta/api_tooling/tooling_api_objects_genaifunctiondefinition.htm): Represents an action that can be added to agents.
-   [GenAiPlannerDefinition](https://developer.salesforce.com/docs/atlas.en-us.api_tooling.meta/api_tooling/tooling_api_objects_genaiplannerdefinition.htm): Represents a planner that uses an LLM and a reasoning strategy to break down a task into smaller parts, find the best actions for each part, and invoke them.
